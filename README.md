# Прогнозирование оттока клиентов кофейного сервиса

## Описание проекта

В рамках проекта разработана модель машинного обучения для прогнозирования оттока клиентов кофейного сервиса.

В ходе работы был выполнен полный цикл построения ML-решения:
- исследовательский анализ данных (EDA);
- предобработка данных;
- анализ пропусков и выбросов;
- анализ корреляций между признаками;
- создание новых признаков;
- построение пайплайна предобработки;
- обучение модели логистической регрессии;
- подбор гиперпараметров;
- оценка качества модели на тестовой выборке;
- сохранение модели для использования в продакшене.

## Итоговая модель

В качестве итоговой модели была выбрана **Logistic Regression**.

В процессе обучения были использованы:

- логарифмирование асимметричных признаков;
- масштабирование числовых признаков;
- One-Hot Encoding;
- Target Encoding для признака `geo_location`;
- подбор гиперпараметра регуляризации `C`;
- удаление малозначимых признаков.

---

## Итоговые метрики

| Метрика | Значение |
|---------|---------:|
| Precision | **0.735** |
| Recall | **0.413** |
| F1-score | **0.529** |
| PR AUC | **0.619** |

Полученная модель значительно превосходит базовую модель (`DummyClassifier`) и способна эффективно выявлять клиентов, склонных к оттоку.
